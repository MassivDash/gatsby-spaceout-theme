## Ardea

Ardea, law from Sopot city in Poland.
Clean and minimalistic look with a bit of elegance. 

Build upon gatsby.js framework with Netlify cms
Fast and simple



## Technology

Ardea business website is a fully customized build based on standard [Gatsby](https://www.gatsbyjs.org/) stater template.   
App written in React framework with [NetlifyCMS](https://www.netlifycms.org/) which provides all the common rich text editing and still enables the website to use the 
jam stack technology. Front end theme was based on [bulma.css](https://bulma.io/) project

This static web app is also equipped with service worker making in a fully downloadable app on the 
android phones (Progressive Web App). Website was deployed via Netlify service.
