## UX / UI 

Projekt mowa is a blog app. The website is mainly designed to boost logopedy and radio voice practice business. 
The client wanted warm colors, clean and minimal layout. I have decided on a rather standard layout blog collection main landing page, 
with some soft white touches and orange accents. Typography and article layout optimized for content delivery, with Roboto as the main body font and Playfair Display for accented messages.

## Technology

Projekt Mowa app was fully customized build based on standard [Gatsby](https://www.gatsbyjs.org/) stater template.   
App written in React framework with [NetlifyCMS](https://www.netlifycms.org/) which provides all the common rich text editing and still enables the website to use the 
jam stack technology. Front end theme was based on [bulma.css](https://bulma.io/) project

This static web app is also equipped with service worker making in a fully downloadable app on the 
android phones (Progressive Web App). Website was deployed via Netlify service.
